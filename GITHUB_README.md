# Repository Scope

This repository is meant to contain:

- the `dinput8.dll` proxy source code
- reverse-engineering notes
- build scripts
- release packaging for the mod itself

It intentionally does **not** include:

- original game binaries
- extracted game assets
- downloaded third-party tooling
- Ghidra project databases

Reason:

- size
- copyright / redistribution risk
- cleaner public repository
